## Setup
Clone the repository make sure you install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install

# bun
bun install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

or

```bash
node .output/server/index.mjs
```
